# AT-UI Style

[![NPM][npm-version-image]][npm-version-url]

`AT-UI Style` is an ui component library based on `Flexbox`, be used for [AT-UI](https://at.aotu.io/)

## Install

```bash
npm install at-ui-style
```

## Usage

```js
import 'at-ui-style'
```
